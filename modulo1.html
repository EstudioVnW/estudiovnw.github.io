HTML5, CSS3, Git e Python

Descrição do curso

Neste módulo você dará o pontapé inicial no universo da programação. Vai aprender a construir e decorar páginas web com HTML e CSS, vai entender as bases do desenvolvimento Front-End, da experiência do usuário e vai criar sites incríveis que se adaptam a todos os tamanhos de tela. Além de conhecer o início do pensamento algorítmico.

Você aprenderá

— Usar linux e outras ferramentas de software livre

— Configurar o ambiente para desenvolvimento

— O que é HTML e CSS

— Inspecionar elementos e resolver bugs

— As propriedades do flexbox

— Introdução ao pensamento algorítmico e processos computacionais

Perfil do estudante

Estudantes com pouca ou nenhuma experiência em programação, com idade preferencialmente entre 16 e 29 anos.
