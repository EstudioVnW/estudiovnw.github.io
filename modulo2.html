Descrição do curso

Entenda como aplicações como o Facebook e o Twitter funcionam e construa você também as suas próprias interfaces interativas para sistemas usando tecnologias modernas, de código aberto, e que grandes empresas de tecnologia também estão usando. Aqui você também aprenderá como começar a contribuir mais ativamente com o desenvolvimento de novas tecnologias de forma colaborativa.

Você aprenderá

— Manipular os elementos de uma página >

— Resolver problemas com algoritmos em Javascript >

— Crie interatividade consumindo APIs públicas >

— Realizar seu primeiro projeto de código aberto >

— Revisão de código >

Perfil do estudante

Estudantes que já concluíram o módulo 1 ou que por já ter experiência em programação passe no teste de nivelamento. Ter preferencialmente entre 16 e 29 anos.
