Descrição do curso

Aprenda como armazenar, organizar e manipular dados, e ainda criar serviços digitais que podem ser consumidos por suas aplicações em diferentes plataformas. Neste módulo você também vai conhecer algumas técnicas para modelar de forma eficiente esses serviços. Essa é a peça final para encaixar no quebra-cabeça, ao final do terceiro módulo você pode juntar todas as peças, digo, conhecimento que você adquiriu, para criar um sistema web completo.

Você aprenderá

— Ética em equipe, código e dados

— Modelagem básica de domínio

— Limites de microsserviços

— Criar uma API simples

— Operações CRUD (Basic SQL e NOSQL)

— Projeto conceitual de banco de dados

Perfil do estudante

Estudantes que já concluíram o módulo 1 e 2 ou que por já ter experiência em programação passe no teste de nivelamento. Preferencialmente entre 16 e 29 anos.
